# Knowledge packs (conteúdo normativo)

O VetorEco **não calcula e não emite ENCE**. A função destes *knowledge packs* é transformar o processo e os manuais relevantes do PBE Edifica em:

- checklist por etapa
- dados mínimos e evidências recomendadas
- explicações curtas ("o que", "por quê", "como")
- referências (links) para as fontes oficiais

Tudo é **versionado** por pacote (ex.: `ini_2025_05`). Quando o PBE Edifica atualizar manuais/notas técnicas, criamos um novo pacote e mantemos o anterior para rastreabilidade.
