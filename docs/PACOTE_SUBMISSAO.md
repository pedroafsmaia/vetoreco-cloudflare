# Pacote de submissão / revisão (exemplo)

O VetorEco não emite ENCE. Ele **te prepara** para entrar no processo oficial com o projeto organizado e com menos retrabalho.

Este documento sugere uma **estrutura de anexos** (nomes/pastas) para você manter o material pronto para a **ENCE de Projeto** e para a **ENCE do Construído**.

> Ajuste conforme o seu escritório, OIA e tipo de edificação.

## Dica prática
O VetorEco permite vincular **evidências** e **cálculos de apoio** a cada item do checklist.
Assim, quando você exporta o dossiê, ele mostra:
- o que está pronto
- o que falta (inclusive “faltando: evidência/cálculo/dados mínimos”)
- e onde estão os anexos
